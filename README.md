# Консольная компьютерная игра «CASINO BLACKJACK» (C++)

Курсовой проект по дисциплине «Программирование на языках высокого уровня». Симуляция классической карточной игры Blackjack в виде автономного консольного приложения.

## 🚀 Основные возможности
* **Честная математическая модель:** Использование генератора `std::mt19937` для абсолютно случайного тасования колоды из 52 карт.
* **Адаптивный псевдографический интерфейс:** Карты отрисовываются в окне терминала с помощью ANSI-последовательностей для управления цветом мастей.
* **Интеллектуальный дилер:** Логика ИИ дилера строго следует правилам казино (обязательный добор карт до достижения 17 очков).
* **Динамический пересчет очков:** Автоматический мягкий и жесткий учет карт типа «Туз» (1 или 11 очков в зависимости от угрозы перебора).
* **Сохранение прогресса:** Автоматическая запись актуального игрового баланса в файл `record.txt` после каждого раунда.

## 📁 Структура проекта
* `main.cpp` — исходный код игры (логика, подсчет очков, рендеринг).
* `Play.command` — bash-скрипт для сборки и запуска приложения на macOS/Linux.
